## Getting Started
Follow these steps to set up and run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Jakaria030/task-one.git
   ```
2. Navigate to the project directory:
   ```bash
   cd task-one
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the command:
   ```bash
   npm run dev
   ```
